## Quick Start

1. Clone the repository:
```bash
git clone https://github.com/yourusername/discord-bot.git
cd discord-bot
```

2. Set up environment:
```bash
cp docker/.env.example docker/.env
# Edit .env with your values
```

3. Run with Docker:
```bash
docker-compose up -d
```
